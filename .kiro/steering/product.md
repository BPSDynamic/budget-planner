# Budget Planner - Product Overview

Budget Planner is a Flutter-based personal finance management application designed to help users track expenses, manage budgets, and analyze spending patterns.

## Core Features

- **Transaction Management**: Record and categorize income and expenses
- **Budget Categories**: Create and manage budget categories with monthly limits
- **Receipt Scanning**: Capture and process receipts via OCR
- **Analytics & Reporting**: View spending trends, forecasts, and historical data
- **Multi-Year Dashboard**: Analyze budget performance across multiple years
- **Settings & Personalization**: Customize currency, language, theme, and notifications
- **User Profiles**: Manage user accounts with profile information

## Target Platforms

- iOS (primary focus with dedicated build/test infrastructure)
- Android
- Web
- macOS
- Linux
- Windows

## Key Characteristics

- Multi-language support
- Dark/light theme support
- Persistent local storage via SharedPreferences
- Provider-based state management
- Material Design 3 UI framework
